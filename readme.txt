Scarypiano is a sample library for Plogue Sforzando version 1.822 or newer, though it may work with other samplers which support SFZ as well. It was made by Karoryfer Samples from the public domain piano samples created by the University of Iowa processed with Krotos Dehumaniser. It was released for Halloween 2015.

To register, drag the Shinyguitar.bank.xml file onto the Sforzando interface. Once registered, Scarypiano will be available in the Sforzando loading menu.

MIDI CC control assignments:
100 - withering (amplitude envelope release length)
101, 102 and 103 - volume controls for the three different voices
